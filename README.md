# ETCH-A-SKETCH #

A browser-based Etch-A-Sketch drawing application built with HTML, CSS, and JavaScript. Users can draw on a dynamic grid, change brush colors, and clear the drawing â€” just like a classic Etch-A-Sketch toy.

# FEATURES #

Draw on grid of hovering squares
Change brush color with buttons
Clear grid at any time
Clear individual boxes with eraser
Adjust brush size dynamically with slider (in Progress)

# Technologies Used #
HTML5 - structure & layout
CSS - Styling & flexbox layout, hover effects
JavaScript - Logic & backend
